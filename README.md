# FitTrack — Premium Fitness & Wellness Platform

## Live Demo
https://fittrackclient.vercel.app/

## Setup
npm install
npm run dev

## Tech Stack
- React 18 + Vite + TypeScript
- Tailwind CSS v3
- Framer Motion
- React Hook Form + Zod
- Zustand
- React Router DOM v6
- Recharts
- Radix UI / Shadcn UI
- Lucide React

## Features
- Landing page with hero, features, testimonials, pricing
- 5-step onboarding flow with validation
- Dashboard with stats, charts, workout tracking
- Dark/light mode toggle
- Fully responsive (375px, 768px, 1440px)

## Architecture
- src/components/ui — reusable base components
- src/components/landing — landing page sections
- src/components/auth — auth and onboarding steps
- src/components/dashboard — dashboard widgets
- src/store — Zustand state management
- src/hooks — custom React hooks
